Japan Data MCP

A Model Context Protocol (MCP) server that gives AI agents cross-source access to Japanese public data — law, corporations, and statistics — normalized to English metadata, with the required source attribution on every response.

Endpoint

MCP over Streamable HTTP (JSON-RPC 2.0):

POST /mcp

A plain GET /mcp returns the server info and the list of available tools (useful for liveness checks).

Tools (Stage 0)

  • search_japanese_law — search Japanese laws and get structured English metadata (no API key required; sourced from e-Gov).
  • search_corporation — look up Japanese corporations by name or corporate number.
  • search_statistics — search official Japanese statistics (e-Stat).

Stage 0 is a minimal, free deployment to measure real demand. Every response carries source attribution. This is an independent data-access utility.
